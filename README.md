# 📊 Projeto de Tratamento, Transformação e Agrupamento de Dados

## 🧾 Descrição
Este projeto tem como objetivo realizar o processamento completo de dados, passando pelas etapas de limpeza (data cleaning), transformação (data transformation) e agrupamento (data grouping/aggregation), com foco em gerar informações organizadas, confiáveis e prontas para análise.

---

## 🎯 Objetivos

### Objetivo Geral
Realizar o tratamento e organização de um conjunto de dados, aplicando técnicas de limpeza, transformação e agrupamento para facilitar análises posteriores.

### Objetivos Específicos
- Identificar e corrigir dados inconsistentes ou ausentes  
- Padronizar formatos e estruturas dos dados  
- Realizar transformações para melhorar a interpretabilidade  
- Agrupar dados com base em critérios relevantes  
- Gerar um dataset final limpo e estruturado

## 🔄 Etapas do Projeto

### 🧹 Limpeza de Dados

### 🔧 Transformação de Dados
